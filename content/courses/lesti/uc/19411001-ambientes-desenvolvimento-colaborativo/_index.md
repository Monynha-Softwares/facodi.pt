## Objetivo

Preparar os alunos para trabalhar em ambientes profissionais de desenvolvimento de software, com foco em ferramentas colaborativas, metodologias ágeis e boas práticas de engenharia de software.

## Conteúdo Programático

### 1. Ambiente de Desenvolvimento
- Ferramentas essenciais (VS Code, Terminal, Package Managers)
- Configuração de máquinas locais
- Debugging e logging
- Integração com serviços em nuvem

### 2. Sistema de Controle de Versão
- Git: Conceitos fundamentais (commits, branches, merge)
- GitHub: Repositórios, pulls requests, issues
- Workflows colaborativos (Git Flow, GitHub Flow)
- Resolução de conflitos

### 3. Metodologias Ágeis
- Scrum: Papéis, cerimônias, artefatos
- Kanban: Fluxo contínuo, WIP limits
- Planning, Daily standup, Sprint review
- Ferramentas: Jira, Trello, GitHub Projects

### 4. Documentação e Comunicação
- README files
- API documentation
- Code comments e docstrings
- Apresentação de projetos

### 5. Colaboração em Equipe
- Code review
- Pair programming
- Communication tools (Slack, Discord)
- Remote work best practices

## Metodologia

- **Seminários práticos**: Trabalho hands-on com ferramentas
- **Projetos em equipa**: Desenvolvimento colaborativo
- **Case studies**: Análise de workflows reais
- **Workshops**: Sessões de aprendizagem ativa

## Avaliação

- Participação em projetos colaborativos: 40%
- Portfólio GitHub: 30%
- Quiz sobre conceitos: 20%
- Apresentação de projeto em equipa: 10%
