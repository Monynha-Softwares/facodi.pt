## Objetivo

Capacitar os alunos na análise e implementação de algoritmos eficientes e estruturas de dados, fundamentais para resolver problemas computacionais complexos e otimizar a performance de aplicações.

## Conteúdo Programático

### 1. Análise de Complexidade
- Notação Big-O, Big-Theta, Big-Omega
- Complexidade temporal e espacial
- Análise amortizada
- Medir performance em prática

### 2. Estruturas de Dados Lineares
- Arrays e listas dinâmicas
- Listas ligadas (simples, dupla, circular)
- Pilhas (stacks)
- Filas (queues)
- Deques

### 3. Estruturas de Dados Hierárquicas
- Árvores binárias
- Árvores de busca binária (BST)
- Árvores balanceadas (AVL, Red-Black)
- Heaps (min-heap, max-heap)
- Tries

### 4. Algoritmos de Ordenação
- Bubble sort, insertion sort, selection sort
- Merge sort, quick sort, heap sort
- Counting sort, radix sort
- Análise e comparação

### 5. Algoritmos de Pesquisa
- Busca linear e binária
- Busca em profundidade (DFS)
- Busca em largura (BFS)
- Busca A*

### 6. Grafos
- Representação de grafos (matriz, lista adjacência)
- Traversal (DFS, BFS)
- Caminhos mais curtos (Dijkstra, Bellman-Ford)
- Minimum spanning tree (Kruskal, Prim)

### 7. Tabelas Hash
- Funções hash
- Resolução de colisões
- Hash tables em Java (HashMap, HashSet)
- Aplicações práticas

## Metodologia

- **Aulas teóricas**: Conceitos com visualizações
- **Laboratórios práticos**: Implementação em Java
- **Análise experimental**: Medição de performance
- **Projetos**: Aplicação em problemas reais
