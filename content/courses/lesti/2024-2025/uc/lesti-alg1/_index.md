Esta unidade curricular apresenta os fundamentos de análise de algoritmos e estruturas de dados lineares e hierárquicas.
Os estudantes aprendem a modelar problemas computacionais, selecionar estruturas adequadas e avaliar a eficiência das soluções.

A componente prática utiliza Python e JavaScript para reforçar a compreensão das abstrações.
